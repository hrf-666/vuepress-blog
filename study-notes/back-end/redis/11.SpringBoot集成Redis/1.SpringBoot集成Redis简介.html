    # 本地Java连接Redis常见问题，小白注意

    1. bind配置请注释掉
    2. 保护模式设置为no
    3. Linux系统的防火墙设置
    4. Redis服务器的IP地址和密码是否正确
    5. 忘记写访问redis的服务端口号和auth密码
